Download

Download Go Server and Go Agent for Free

You need to download both the server and the agent in order to use Go.

Scripting Go installations gets easier now. Both Go server and agent are now available on public YUM and APT repositories!

If you need help downloading, installing or using Go please join the Go Users Google Group. ThoughtWorks also provides commercial help for Go. They can be reached at support@thoughtworks.com.

New! Subscribe to updates on new Go CD releases and more.

Upgrading? Go 15.2.0 does not work with Java 6. Please make sure you use at least Java 7 (preferably the latest stable version) on the Go Server and Go Agents.
